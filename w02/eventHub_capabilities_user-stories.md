# EventHub — An Event Ticket Booking Platform

This is an online event ticket booking system.

## Core Capabilities
EventHub allows:
- **Event Organizers** to create and manage events (concerts, conferences, workshops)
- **Customers** to browse events, select seats/tickets, and complete purchases
- **Administrators** to oversee the platform, manage users, and view analytics

## Key User Stories
| ID | As a... | I want to... | So that... |
|----|---------|-------------|------------|
| US-01 | Customer | browse upcoming events with filters | I can find events I'm interested in |
| US-02 | Customer | select tickets and complete a purchase | I can attend the event |
| US-03 | Customer | receive a QR-code ticket via email | I have proof of purchase |
| US-04 | Customer | cancel my booking within 48 hours | I can get a refund if plans change |
| US-05 | Organizer | create and publish an event | people can discover and book it |
| US-06 | Organizer | view real-time sales data | I can track how my event is performing |
| US-07 | Admin | manage users and roles | the platform stays secure and organized |
| US-08 | Admin | view platform-wide analytics | I can make business decisions |
